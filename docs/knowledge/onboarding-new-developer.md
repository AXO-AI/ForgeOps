# New Developer Onboarding

## Day 1
1. Get GitHub account added to the org
2. Install VS Code + extensions: GitHub Actions, Copilot, Pull Requests
3. Clone your app repo
4. Create a feature branch: `git checkout -b feature/JIRA-XXX-my-task int`

## Day 2
1. Make changes, commit with Jira key in message
2. Push branch → watch CI pipeline run in Actions tab
3. Create PR to int from dashboard
4. Ask Tech Lead to review
5. After approval → merge → deploys to INT automatically
