# Configure Jira
1. Sign up: atlassian.com/software/jira/free
2. Create token: id.atlassian.com → Security → API Tokens
3. Add secrets: JIRA_URL + JIRA_TOKEN in GitHub Org → Settings → Secrets
4. Create Jira statuses: Ready for Unit Testing, Unit Testing Complete, Ready for SIT, SIT Complete, Ready for UAT, UAT Complete, Deployed to Production
